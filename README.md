# 港澳新导师信息库 (Prof-List for CN Students)

[![LICENSE](https://img.shields.io/badge/license-CC--BY--4.0-blue.svg)](LICENSE)

这是一个开源项目，旨在为中国学生（尤其是受10043等政策影响的学生）提供一个集中的、可更新的香港、澳门、新加坡等地区的导师信息数据库。

所有数据均存放在 `prof_list.csv` 文件中。

[简述10043政策带来的困难，以及本项目如何帮助学生节省时间、精准定位]

## 如何使用？

1.  **直接浏览：** 你可以直接在 GitHub 上点击 `prof_list.csv` 文件进行在线浏览。
2.  **下载使用：** 点击 `prof_list.csv` 文件，然后点击 "Download raw file" 按钮，将其下载到本地用 Excel 或其他表格软件打开。
3.  **结合 LLM (推荐)：** 下载 `.csv` 文件后，你可以将该文件上传给 ChatGPT (GPT-4), Gemini, Kimi, Claude 等大语言模型工具，然后结合你自身的情况（如研究兴趣、GPA等）进行提问，例如：
    > "这是港澳新地区的导师列表，我的研究方向是计算机视觉和深度学习，请帮我筛选出所有相关的教授，并按职称排序。"

## 数据结构
`prof_list.csv` 文件包含以下 7 个字段：
`地区,学校,老师,大方向,职称,细化方向,主页`

本项目依赖社区共同维护！你的贡献至关重要。我们提供两种贡献方式：

### 方式一：填写表单 (推荐，适合所有人)
这是最简单、最推荐的方式。你不需要懂 Git 或编程。

➡️ **[点击这里提交新教授信息](https://YOUR_GOOGLE_FORM_LINK)**

你提交的信息会进入审核队列，维护者会定期审核并将其添加到 `prof_list.csv` 主文件中。

### 方式二：提交 PR (适合熟悉 GitHub 的用户)
如果你熟悉 Git 和 GitHub，我们欢迎你通过 Pull Request (PR) 直接修改 `prof_list.csv` 文件。

**在贡献之前，请务必阅读我们的 [贡献指南 (CONTRIBUTING.md)](CONTRIBUTING.md)** 以了解详细的数据格式标准。
